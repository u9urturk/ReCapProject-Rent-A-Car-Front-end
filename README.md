## Installation

```bash
# Clone to repository
$ git clone https://github.com/u9urturk/ReCapProject-Rent-A-Car-Front-end
  
  
# Go to the folder you cloned
$ cd ReCapProject-Rent-A-Car-Front-end
  
# Project started with VSCode
$ code .

# Install dependencies
$ npm install
```
